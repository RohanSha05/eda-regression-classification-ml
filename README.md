# eda-regression-classification-ml
End-to-end machine learning workflow including exploratory data analysis, outlier detection, feature preprocessing, linear regression, and classification modeling with performance evaluation and visualizations using Python and scikit-learn.
## Overview
This project demonstrates core machine learning concepts including EDA,
data preprocessing, regression, and classification.

## Sections
- Setup and library imports
- Section A: Helper functions for descriptive statistics and outlier detection
- Section B:
  - Exploratory Data Analysis (EDA)
  - Data preprocessing (imputation, encoding, scaling)
  - Linear regression modeling and evaluation
  - Classification metrics, ROC analysis, and decision tree modeling

## Tools & Libraries
Python, NumPy, pandas, matplotlib, scikit-learn
